# 启用 Yarn 支持

提示

如果这一步遇到问题，请看 yarn 文档 (opens new window)

corepack enable
1

注意

这一步可能需要使用 UAC ，即使用管理员身份打开 PowerShell 窗口运行以上命令。

# 安装依赖

yarn install
1

注意

执行以上命令时可能会遇到无法执行无签名 powershell 脚本的问题，详见 关于执行策略 (opens new window)

需执行

Set-ExecutionPolicy Bypass
1

or

Set-ExecutionPolicy AllSigned
1

提示

安装过程中如果无法稳定连接 yarn 官方仓库，可启用国内镜像源，详见 配置 章节。

# 测试和预览

yarn dev
1

# 本地构建

yarn build
